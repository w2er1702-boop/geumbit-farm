# 금빛농원 홈페이지 시안

정적 HTML/CSS/JS로 만든 금빛농원 브랜드/판매 홈페이지 시안입니다.

## 페이지

- `index.html`: 홈
- `story.html`: 브랜드 스토리
- `products.html`: 제품 목록
- `product-detail.html`: 제품 상세
- `reviews.html`: 후기
- `community.html`: 커뮤니티
- `cart.html`, `login.html`, `join.html`, `mypage.html`: 커머스 모듈 시안
- `support.html`, `guide.html`, `terms.html`, `privacy.html`: 고객센터 및 정책

## 배포

GitHub Pages에서 `main` 브랜치의 루트(`/`)를 소스로 지정하면 바로 배포할 수 있습니다.
