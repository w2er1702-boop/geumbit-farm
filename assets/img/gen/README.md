# Generated Sanghwang Images

These images are original AI-generated staging assets for Geumbit Farm.

They were created after reviewing staging categories and user-provided sanghwang mushroom references, but reference photos were not copied into the public site. The prompts intentionally use different compositions, props, backgrounds, and lighting while preserving the factual visual traits of yellow sanghwang mushroom:

- bright yellow whole sanghwang chunks
- rounded, lumpy, irregular surfaces
- sliced yellow-brown strips with darker crust edges
- real farm handling, soaking, and cultivation-log contexts
- woven bamboo/rattan trays and warm wooden table settings

Website pages use the optimized `.jpg` files. The `.png` files are preserved as higher-quality generation copies.
